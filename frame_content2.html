HTML Frame Attributes

Both the <frameset> and <frame> tags have various attributes that control how frames are displayed and behave. Understanding these attributes is essential for working with HTML frames.

Frameset Attributes

The <frameset> tag defines how to divide the window into frames. It replaces the <body> tag in frameset documents.

Attribute Description Example
rows Specifies the height of each row in the frameset. Values can be in pixels, percentages, or relative sizes using asterisk (*) rows="100,*,200"
cols Specifies the width of each column in the frameset. Values can be in pixels, percentages, or relative sizes using asterisk (*) cols="25%,75%"
border Specifies the width of the border between frames (in pixels) border="5"
frameborder Specifies whether or not to display borders between frames (1=yes, 0=no) frameborder="0"
framespacing Specifies the amount of space between frames (in pixels) framespacing="10"

Example of Frameset with Attributes:

<frameset rows="100,*" cols="25%,75%" border="5" frameborder="1" framespacing="10">

Frame Attributes

The <frame> tag defines a single frame within a frameset. It specifies which HTML document should be displayed in that frame.

Attribute Description Example
src Specifies the URL of the document to display in the frame src="menu.html"
name Assigns a name to the frame (used as the target for links) name="content"
frameborder Specifies whether or not to display a border around the frame (1=yes, 0=no) frameborder="0"
marginwidth Specifies the left and right margins inside the frame (in pixels) marginwidth="10"
marginheight Specifies the top and bottom margins inside the frame (in pixels) marginheight="10"
noresize Prevents the user from resizing the frame noresize
scrolling Specifies whether or not to display scrollbars in the frame (yes, no, auto) scrolling="no"
longdesc Specifies a URL to a document containing a long description of the frame's content longdesc="framedesc.html"

Example of Frame with Attributes:

<frame src="content.html" name="mainFrame" frameborder="0" marginwidth="10" marginheight="10" scrolling="auto" noresize>

Using Frame Names with Links

One of the most important attributes is the name attribute, which allows you to target specific frames with links. This is done using the target attribute in anchor tags.

Example of Targeting Frames with Links:

In a navigation frame, you might have links like this:

<a href="page1.html" target="mainFrame">Page 1</a>

This link would load "page1.html" in the frame named "mainFrame".

Special Target Names

There are also special predefined target names that have specific behaviors:

Target Name Description
_self Loads the page in the same frame as the link
_parent Loads the page in the parent frameset
_top Loads the page in the full browser window, replacing all frames
_blank Loads the page in a new browser window or tab
