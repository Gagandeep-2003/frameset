Introduction to HTML Frames

Welcome to this educational demonstration of HTML frames. This page is designed to show how frames work in HTML, even though they are deprecated in modern web development.

This webpage is divided into three frames:

  • Top Frame: Contains the header
  • Left Frame: Contains the navigation menu
  • Main Frame: Contains the main content (you're viewing this now)

What are HTML Frames?

HTML frames allow developers to divide the browser window into multiple sections, where each section can load a separate HTML document. This was commonly used for creating layouts with fixed navigation bars and headers.

The main HTML tags used for frames are:

Tag Description
<frameset> Defines a set of frames
<frame> Defines a single frame within a frameset
<noframes> Provides content for browsers that do not support frames

Basic Frameset Structure

Here's the basic structure of the frameset used in this example:

<frameset rows="80,*">
    <frame src="frame_top.html" name="topFrame" scrolling="no" noresize>
    
    <frameset cols="200,*">
        <frame src="frame_left.html" name="leftFrame">
        <frame src="frame_main.html" name="mainFrame">
    </frameset>
    
    <noframes>
        <body>
            <p>Your browser does not support frames.</p>
        </body>
    </noframes>
</frameset>

Important Note:

Frames are deprecated in HTML5 and should not be used in modern web development. They have several disadvantages:

  • Poor accessibility
  • SEO problems (search engines have difficulty indexing framed content)
  • Bookmarking issues
  • Printing problems

Modern alternatives include CSS layouts, iframes (for specific use cases), and JavaScript-based solutions.
